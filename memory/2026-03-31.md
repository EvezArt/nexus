## 11:44 — NEXUS: Unified 24/7 Chatbot (Steven)
Steven asked for "plugins that connect ChatGPT and Perplexity chats and memory into one continuous 24/7 chatbot that can outdo OpenClaw or even use OpenClaw."

Built the complete NEXUS system:

### Architecture
```
User ↔ Nexus Core ↔ Provider Router
                    ├── ChatGPT (reasoning, code, creative)
                    ├── Perplexity (search, research, citations)
                    └── OpenClaw (local spine, daemon, cognition)

All providers write to → Unified Memory (spine + daily logs + index)
All providers read from ← Unified Memory (context injection)
```

### Components (8 files)
- `nexus/__init__.py` — package init
- `nexus/providers/base.py` — Message, ProviderResponse, BaseProvider ABC
- `nexus/providers/chatgpt.py` — OpenAI ChatGPT API adapter (httpx, no openai package needed)
- `nexus/providers/perplexity.py` — Perplexity API adapter with citation extraction
- `nexus/providers/openclaw_bridge.py` — bridges to local morpheus spine + cognition
- `nexus/memory_store.py` — unified memory: index + daily logs + spine + search + context retrieval
- `nexus/nexus_core.py` — orchestrator: smart routing by query type, fallback chains, memory injection
- `nexus/nexus_daemon.py` — 24/7 daemon: polling or HTTP server mode, chat queue, auto-recovery
- `nexus/nexus_ctl.py` — CLI: start/stop/status/chat/research/config/logs

### Smart Routing
- Research queries → Perplexity (web search + citations)
- Code/reasoning → ChatGPT (strong inference)
- Routine/status → OpenClaw (local, fast, no API cost)
- Learns from outcomes via exponential moving average

### Key Insight
Memory is provider-agnostic. A ChatGPT conversation and a Perplexity search produce the same memory format. This is how the nexus maintains continuity across provider switches — any provider can pick up where another left off because they all share the same memory backbone.

### What It Needs
- API keys: `python3 nexus/nexus_ctl.py config set chatgpt_api_key "sk-..."`
- API keys: `python3 nexus/nexus_ctl.py config set perplexity_api_key "pplx-..."`
- Without keys, OpenClaw (local cognition) is the only provider

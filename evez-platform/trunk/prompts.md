## Surface Sequence (Default)
1. **Perplexity** = gather evidence
2. **ChatGPT** = challenge the logic
3. **Claude** = refactor the surviving structure
4. **Base44/n8n** = turn the approved structure into persistent operator behavior
5. Return the new state to trunk

## Auto-Advance Rules
1. If a branch finishes cleanly, spawn the next branch automatically.
2. If two branches disagree, route both through Skeptic before asking the human.
3. Every 4 branches, compress the best surviving logic into the trunk state.
4. Only surface decisions that are irreversible, external-facing, or capital-committing.
5. Preserve one canonical state so all surfaces are building the same system.

## Branch Contract (Return Structure)
Every branch returns:
1. Objective
2. Assumptions
3. Output
4. Failure modes
5. Confidence (0.0-1.0)
6. Return-to-trunk summary
7. Next automatic branch

## How to Deploy

```bash
# Clone the template
git clone https://github.com/EvezArt/evez-agent-template-<name>.git

# Configure
cd evez-agent-template-<name>
cp .env.example .env
# Edit .env with your API keys

# Deploy
./deploy.sh
```
